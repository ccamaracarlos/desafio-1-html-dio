Comunicação Client x Server

    Sobre Clients

    Voltar ao Indice

    O termo Client é a forma de pensar pelo ponto de vista do cliente, tudo que se refere a
    um usuário, podemos dizer que é a visão de Cliente

    O Client é a propria aplicação em si, podemos pensar em Client como o usuário acessando a rede atravez de um navegado ou App.

    Sobre Navegadores

    Navegadores são programas para acessar a internet atravez do seu dispositivo.
    Tambem são conhecidos como Browser

    Alguns exemplos de navegadores são:

    • Safari ========= da Apple
    • Firefox ======== da monzila
    • Chrome ======= da Google
    • Internet Explore = da Microsoft
    • Edge ========= da Microsoft

    Sobre Aplicações Web

    Aplicações Web nada mais são que softwares que são desenvolvidos e hospedados dentro de servidores de aplicações
    que possui domínio e um DNS

    Esse software pode ser acessado atravez do protocolo http.
    Você pode ter mais de uma aplicação em um único servidor de aplicação

    Voltar ao Indice

    Sobre Server

    O termo Server é indicado quando nos referimos a tudo que acontece nos servidores
    que por sua vez é responsável por receber as requisições do client processar essa requisição e devolver para o client

    Para ficar mais claro, podemos exemplificar.
    Um usuário clica para baixar uma imagem o servidor recebe essa solicitação, processa a mesma e devolve a imagem para o usuário.

    Voltar ao Indice

    Alguns tipos de Servidores

    Um servidor é uma maquina mais robusta, ele tem o papel de armazenar e distribuir dados para toda rede que o utiliza.
    Ele faz o gerenciamento de usuários da rede.
    Controle de acesso.
    Compartilhamento de arquivos e dispositivos

    Tipos de Servidores:

    • Servidor de arquivos
    • Servidor Web
    • Servidor de Impressão
    • Servidor de banco de dados

    Cada servidor tem sua função de acordo com a necessidade do usuário.

    Linguagem de Programação

    As linguagens de Programação são utilizadas como comunicação entre programador e maquina.
    Geralmente utilizamos as linguagens de programação para fazermos aplicações Web, na maioria das vezes atuando na lógica da aplicação, conhecida como Back-end.

    Algumas linguagens de programação que utilizamos como Back-end:

    • JAVASCRIPT
    • PHP
    • COBOL
    • JAVA

    Fato curioso é que a linguagem JAVASCRIPT, pode ser utilizada tanto no back-end como no Front-end.

    Falando sobre Tags

    As Tags são etiquetas que informamos ao navegador como o site deve ser apresentado.
    Utilizamos as tags para marcar o início de um bloco de comandos e o fim desse bloco, atravez do sinal de menor (<) e maior (>)

    Cada tag tem seu propósito, como descritos a baixo:

    • Tag (a)------------ Significa que queremos ancorar, colocar um link no elemento
    • Tag (p)------------ Significa um paragrafo
    • Tag (ol) ---------- Significa que vamos montar uma lista ordenada

    Assim como estes exemplos existem várias outras tags a serem utilizadas, tudo vai depender da necessidade da aplicação.

    Voltar ao Indice

    Atributo das Tags

    Os atributos nas tags são de suma importancia, a maioria das tags que utilizamos no html tem seus atributos e valores.
    Os atributos podem mudar a parencia e o comportamento dos elementos

    Quando utilizamos a tag (a) significa que queremos colocar um link no elemento. Nesse caso, o atributo da tag a sera o (href).
    Também temos os atributos (class) e (id), dentre outros.
